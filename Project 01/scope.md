## Purpose
The purpose of this project is to create a basic introduction webpage for CMPA 3301 that shows I understand how to use HTML and organize information in a clean, simple way.

## Description
This project includes two parts: a scope statement and an HTML file. The scope statement explains what the project is about, what I plan to include, and what is not part of the project. The HTML file will be a short webpage about me. It will have my name, a short introduction, my goals for the course, my career interests, a list of hobbies, and a link to my GitHub profile. The page will only use the required HTML tags and will focus on structure, not design.

## Desired Results
- A complete index.html file that uses the correct semantic tags.
- A webpage that clearly introduces who I am and what I’m studying.
- A working GitHub link included in the page.
- A scope.md file that follows the PMUPM Chapter 3 format.
- Both files saved together and submitted as one .zip file before the deadline.
- A project that shows I understand how to structure and organize content using HTML.

## Exclusions
- No CSS or styling.
- No JavaScript.
- No images.
- No extra pages.
